# Project: You wrote x, y, and z.

## Description
* This program lets user enter an input for 3 variables named x, y, and z.
* The program prints the same 3 variables back into the terminal.

This project was built as an exercise for **LearnCPP [Chapter 1, lesson 5, [question #2] aka 1.5]**.

## Structure of the program
* Header files[Numbers.h]:- This file contains the declaration of function numbers()
* Source files[Numbers.cpp]:- This file contains the definiton of function numbers()
* Main files[Main.cpp]:- This file declares Numbers.h at line 2, then function calls numbers() inside main() function.
