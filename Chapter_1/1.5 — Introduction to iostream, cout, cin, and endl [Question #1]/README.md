# Project: You wrote x

## Description
* This program tells and allows users to enter a number and after receiving an input, the program prints the same input back into the terminal.

This project was built as an exercise for **LearnCPP [Chapter 1, lesson 5, [question #1] aka 1.5]**.

## Structure of the program
* Header file [Numbers.h]:- This file declares function number().
* Source file [Numbers.cpp]:- This file contains definiton of function number().
* Main file [Main.cpp]:- In this file function number() is declared at line 2. Then inside main function, function number() is called.
