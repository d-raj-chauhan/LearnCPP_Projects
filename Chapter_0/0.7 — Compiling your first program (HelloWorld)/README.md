# Hello World!

## Description
* This is a very short and simple project named "Hello World!". 
* In this project, we print the text "Hello World!" to the terminal.

This project was built as an exercise for **LearnCPP [Chapter 0, lesson 7 aka 0.7]**.

## Structure of the program
* For this project, I wrote code inside my main.cpp file, since printing "Hello World" didn't take that much space.

## What I Learned
Working on this project helped me practice:
* [New] Concept 1, e.g., Basic standard input/output using std::cin and std::cout
* [New] Concept 2, e.g., How to use main function
* [New] Concept 3, e.g., What std::cin.get(); does?

## How to Compile/Run
This is a standard C++ console application. It can be compiled using any standard C++ compiler (like GCC, Clang, or MSVC).
