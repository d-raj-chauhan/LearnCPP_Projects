# Project: Addition and Subtraction

## Description
* This program lets user enter an input for 2 variables of data type int named firstInteger, and secondInteger.
* The program then gives the addition and subtraction of those same integers and prints them into the terminal.

This project was built as an exercise for **LearnCPP [Chapter 1, lesson x, [question #3] aka 1.x**.

## Structure of the program
* Header files[Integer.h]:- This file declares the function number()
* Source files[Integer.cpp]:- This file contains the definition of function number()
* Main files[Main.cpp}:- This file declares Integer.h at line 2 to use number().
